# Git::Peer

Peer into your release diffs--between git refs--with the greatest of ease.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'git-peer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git-peer

## Usage

`git peer x.y.z`
